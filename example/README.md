# Dot JS Example App

## How to run

1. Install dependencies:
   ```
   npm install
   ```
2. Start dev server:
   ```
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Features

- State management with localStorage
- Routing
- Reusable components
- Event delegation
- HTTP requests
- Performance demo (virtual list)
